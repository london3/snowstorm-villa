# 雪嵐の山荘 ~ A Night Without Tomorrow ~

Webで公開できる、ライトノベル風の短編ミステリーノベルゲームです。
プレイヤーは主人公に名前を付け、雪嵐に閉ざされた山荘で起こる殺人事件の真相を、選択肢を通じて追っていきます。

## 特徴

- **約20分で完結** する短編ストーリー
- **ハッピーエンド（TRUE END）／バッドエンド（BAD END）** の分岐
- **主人公の名前入力** に対応（本文中に名前が反映されます）
- **3スロットのセーブ／ロード** （`localStorage` を使用）
- **シーンに応じたBGM** （穏やか／不穏／恐怖／クライマックス／ハッピー／バッド の6種）を WebAudio API でリアルタイム生成
- **事件発生時のショッキングな効果音** （ノイズ＋低音＋悲鳴グリッサンド）＋画面のフラッシュ／シェイク
- スマートフォン対応のレスポンシブUI

## 遊び方

1. `index.html` をブラウザで開く
2. タイトル画面で `新しく始める` を選ぶ
3. 主人公の名前を入力して開始
4. 本文を読み進め、選択肢で物語の運命を決定する
5. ヘッダーの `セーブ` `ロード` ボタンでいつでも保存・復元できる
6. `♪ ON / OFF` ボタンで BGM・効果音の有効／無効を切り替え

## 公開方法

このゲームは静的ファイル（HTML/CSS/JS）のみで動作するので、
GitHub Pages / Netlify / Vercel / Cloudflare Pages などの静的ホスティングにそのまま配置できます。

```
index.html
style.css
game.js
```

の3ファイルをそのままアップロードするだけです。

## カスタマイズ

- 物語の本文・分岐は `game.js` 内の `STORY` オブジェクトに集約されています。各シーンは
  - `chapter`: 画面上部の章タイトル
  - `mood`: BGMの雰囲気 (`calm` / `tense` / `horror` / `climax` / `happy` / `bad`)
  - `pages`: 順に表示される本文（`{name}` は主人公名に置換）
  - `shock`: ショッキングSEを再生するページ番号の配列
  - `choices` または `next`: 次への分岐
  - `ending`: `'happy'` または `'bad'` の場合は終幕
  という構造で、自由に追加・編集できます。
- BGMやSEは `buildBGM` / `playShockSE` で WebAudio から生成しているため、外部ファイル不要です。
- セーブデータは `localStorage` の `storygame_slot_1` ～ `storygame_slot_3` に保存されます。
